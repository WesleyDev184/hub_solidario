Política de Privacidade - Sistema de Gestão Hub Solidário

Esta Política de Privacidade descreve como o Sistema Hub Solidário de Gestão de Empréstimos de Órteses ("Sistema") coleta, usa, armazena e protege suas informações pessoais. Ao utilizar o Sistema, você concorda com as práticas descritas aqui. O Sistema é operado pelo Hub Solidário e tem como objetivo gerenciar empréstimos de órteses para fins sociais e assistenciais.

1. Introdução

Esta Política de Privacidade descreve como o Sistema Hub Solidário de Gestão de Empréstimos de Órteses ("Sistema") coleta, usa, armazena e protege suas informações pessoais. Ao utilizar o Sistema, você concorda com as práticas descritas aqui. O Sistema é operado pelo Hub Solidário e tem como objetivo gerenciar empréstimos de órteses para fins sociais e assistenciais.

2. Dados Coletados

Coletamos os seguintes tipos de dados pessoais:

  • Dados de Usuários (Administradores/Operadores): Nome, email, telefone, senha (criptografada), hub (banco ortopédico) associado e tokens de autenticação.
  • Dados de Requerentes (Applicants): Nome, CPF, email, telefone, endereço, status de beneficiário, quantidade de beneficiários, imagem de perfil e hub associado.
  • Dados de Dependentes: Nome, CPF, email, telefone, endereço, imagem de perfil e vínculo com o requerente.
  • Documentos: Arquivos enviados (como comprovantes ou laudos médicos), associados a requerentes ou dependentes.
  • Dados de Estoque e Empréstimos: Informações sobre itens (códigos seriais, estados), estoques (títulos, quantidades), empréstimos (datas, responsáveis) e hubs (nomes, cidades).
  • Dados Técnicos: Logs de acesso, endereços IP, dados de cache para otimização de performance e tokens de sessão.

Esses dados são coletados via formulários no aplicativo, uploads de arquivos e interações com a API.

3. Uso dos Dados

Utilizamos seus dados para:

  • Gerenciar empréstimos de órteses e controlar estoques.
  • Autenticar usuários e autorizar acessos.
  • Processar cadastros de requerentes, dependentes e documentos.
  • Melhorar a performance do Sistema através de cache (local e em nuvem).
  • Cumprir obrigações legais e assistenciais do Hub Solidário.
  • Enviar notificações relacionadas a empréstimos ou manutenções.

Não utilizamos os dados para fins comerciais ou marketing sem consentimento explícito.

4. Compartilhamento de Dados

Seus dados não são vendidos ou compartilhados com terceiros, exceto:

  • Com provedores de serviços essenciais (ex.: armazenamento em nuvem como S3 para imagens/documentos, Redis para cache).
  • Quando exigido por lei ou para proteger direitos legais.
  • Com membros autorizados do Hub Solidário para fins operacionais.

Todos os compartilhamentos seguem princípios de necessidade e segurança.

5. Segurança dos Dados

Implementamos medidas de segurança, incluindo:

  • Criptografia de senhas e dados sensíveis.
  • Controle de acesso baseado em autenticação (API Key e tokens Bearer).
  • Armazenamento seguro em bancos de dados PostgreSQL e cache Redis.
  • Auditoria de logs para detectar acessos não autorizados.
  • Exclusão automática de arquivos temporários e limpeza de cache expirado.

No entanto, nenhum sistema é 100% seguro; recomendamos o uso de senhas fortes e não compartilhamento de credenciais.

6. Cookies e Tecnologias Similares

O Sistema utiliza cache híbrido (memória local e Redis) para otimizar performance. Não utilizamos cookies tradicionais, mas armazenamos tokens de sessão localmente no dispositivo para manter o login ativo.

7. Seus Direitos

Você tem o direito de:

  • Acessar, corrigir ou excluir seus dados pessoais.
  • Solicitar portabilidade dos dados.
  • Revogar consentimento para processamento não essencial.
  • Entrar em contato para dúvidas ou reclamações.

Para exercer esses direitos, entre em contato conosco via email de suporte.

8. Retenção de Dados

Retemos dados apenas pelo tempo necessário para os fins descritos. Dados de empréstimos são mantidos por até 5 anos após o término, conforme regulamentações assistenciais. Dados de usuários inativos podem ser excluídos após 2 anos de inatividade.

9. Alterações nesta Política

Podemos atualizar esta Política periodicamente. Notificaremos sobre mudanças significativas via aplicativo ou email. A versão mais recente estará disponível no Sistema.
